Detailed Description

template<typename T, class Context, class Reducer, bool FirstDim, class InputAccessor = BaseInputAccessor<T>>
class caffe2::AbstractReduceFrontOrBackOp< T, Context, Reducer, FirstDim, InputAccessor >

Simple non-segmented reduction over the first few dimensions of the tensor.

Inputs: 0: DATA - input embedding to do lookups in 1..P: AUX_ARG_ - optional additional arguments to be passed to the reducer

Args: num_reduce_dim (default 1) - the number of dims in front of the tensor to reduce

Output: Tensor without the first num_dim dimensions of DATA
